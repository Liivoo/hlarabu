64. THERE IS A FOUNTAIN (S.S.129)

“…a fountain opened…for sin and for uncleanness.” (Zech.13:1)

WILLIAM COPPERDoh is COLD MELODY

1) Thisen tuidung inkhat kalōng,
Christa Ratha hrui ah;
Set intheng nasik khat kaom,
Set rakip intheng jo.

Set rakip Amantheng,
Set rakip Amantheng;
Ni setna Amantheng sō jo,
Set rakip intheng jo.

2) Set karôn hei inpâk jo hei,
Hring na an hmu jârra;
Kei um set karôn eng kaeh,
Kaset amantheng sik.

3) Eini’n aṭong ni jôm inchu,
Amalungset sik me;
Mahngi mak ning alungset na,
Rōngpu’n alungset me.

4) Ṭhat na rakip mahngi mak ning,
Nasōk in marvân âm;
Kabei innum manpâk che rong,
Manpâk hla sak ru-ung.
