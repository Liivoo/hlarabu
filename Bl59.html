59. HE IS ABLE TO DELIVER THEE (S.S.848)

“…our God whom we serve is able to deliver us….” (Dan.3:17)

W.A.O.Doh is BbWILLIAM A. OGDEN

1) Set kahrik karôn mihei seng kha,
Rōngpu set thina re ake tor pui hei;
Ti âm ni bei âm la hri phit mak me,
Hla vâng um lasak manphut jên mak me.

Jesûn re ake tor pui sik che,
Rōngpu’n re ake tor pui sik che;
Set nanei inchu Ahneng ah se roh,
Amân hei rakip amantheng sō thei.

2) Vânhnōi ralei tui-ōng pui chung ah,
Mihring set tui kalên sûng ah kaom;
Anhneng ah Rōngpu ṭong kha vahri roh,
Rōngpu’n re amanke sōt tor sik hei.

3) Set jârra rathla khōdeng kathei,
Rōngpu taksenna nanei êt inchu;
Nan rathla inreng amanhōi sō sik,
Rōngpu’n re ake sōt pui tor sik me.
